AI Prompt ToolKit for SWE

A collection of reusable prompt templates that I've designed to assist with clarity, decision-making, and efficiency when working with Claude, Gemini, and ChatGPT, however they can be reused with other AI tools and workflows. 

Purpose 
This repository helps to : 
- Reduce ambiguity when using AI 
- Improve output and quality with structured prompts. 
- Encourage planning before coding 
- Provide reusable templates for engineering. 

Method : 
First plan, then code 

Use key phrase to trigger code : 
Ex : "Let's Dance" 
- This will unlock implementation. 
Simplicity is preferred over Complexity 
- Minimal Solutions
- Avoid unnecessary abstractions 
Resource Discipline 

- Max 2-3 links per prompt 
- Only high-value documentation links, it should filter out redundancies 

Working Tree : 
- prompts/ 
- resources/ 
- README.md

How to use : 
1. Copy a prompt template
2. Fill in your context 
3. Run with AI
4. Stay in planning mode
5. Type key phrase : "Let's Dance", or phrase of choice

Mode : 
What --> tasks
How --> constraints 
Why --> priorities 
Scope --> boundaries
What NOT to do --> Prevent Complexities 

These templates specifically focus on 
- Clarity 
- Speed 
- Real World Use Cases
