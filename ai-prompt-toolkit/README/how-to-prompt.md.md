#  How to Use Your AI Prompt Toolkit (Beginner Guide)


This toolkit helps you use AI in a **smarter and more structured way**.

# ⚙️ The Simple Way to Use This

Follow this 5-step process every time:

## 1. Pick the Right Prompt

Choose based on what you’re doing:

- Something broke → _Incident Response_

- Debugging → _Log Analysis_

- Security → _IAM / Security Prompt_

- Scaling → _Capacity Planning_

- Deploying → _DevOps Checklist_

- Designing → _Architecture Prompt_


## 2. Run the Prompt using Planning Mode

Paste the prompt into AI and **DO NOT say**:

```text
Let's Dance
```

You will get:

- ideas
- structure
- risks
- suggestions

*This is the thinking phase*

## 3. Write Down Key Decisions

*Important : Summarize for understanding*

Example:

```text
Decisions:
- Use simple deployment strategy
- Add logging for errors
- Avoid complex architecture
```

## 4. Use your decisions : 

- a checklist
- a TODO list
- notes in your README
- a simple plan

Example:

```text
TODO:
- Add error handling
- Improve UI clarity
- Add logging
```

## 5. Execute (Only When Ready)

When you're ready to actually build or fix something, type:

```text
Let's Dance
```

Now AI will give:

- commands
- code
- configs

# Safety Rules 
*Security checks are built into the templates, double check for errors*
Always follow these:

- Never share passwords or API keys
- Never paste full raw logs
- Use placeholders like `[REDACTED]`
-  Focus on patterns, not sensitive data

If something looks risky:

```text
⚠️ SECURITY WARNING: Potential sensitive data exposure detected
```

# Simple Mental Model

Every prompt helps you answer:

- What am I doing?
- How should I approach it?
- Why does it matter?
- What should I avoid?
