#  Network & System Administration Prompts

This folder contains reusable templates for network and system admin task, for troubleshooting, system design and health checks. 

##  Purpose

These prompts are designed to:

- Provide **structured troubleshooting approaches**

- Improve **system reliability and stability**

- Encourage **safe, low-risk decision-making**

- Support **real-world operational workflows**

- Reduce guesswork when diagnosing issues

##  How to Use

1. Copy a prompt template
2. Fill in your system or issue details
3. Edit any field to better match your concern
4. Run it with your AI tool, (tested in ChatGPT & Gemini & Claude) 
5. Stay in planning mode
6. Use:


```text
Let's Dance
```

to unlock commands, scripts, or configuration steps

## Design 

### Stability First

- Prioritize uptime and reliability
- Avoid risky or irreversible changes
### Safe Iteration

- Recommend reversible actions
- Identify risks before execution

### Structured Thinking

- Separate investigation from solution
- Follow clear diagnostic steps

### System Admin Checklist

Use for:

- reviewing system health
- validating configurations
- performing pre-deployment checks

##  Mental Model

Each prompt follows this framework:

- **What** → system or issue
- **How** → structured investigation or plan
- **Why** → reliability, security, and performance
- **Scope** → practical operational boundaries
- **What NOT to do** → avoid risky or premature actions

## Important Notes

- Prompts default to **safe, non-destructive recommendations**
- Commands and scripts are **not generated** until triggered
- Focus is on **understanding before acting**
